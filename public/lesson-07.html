第七课 · 模型对比

国产 vs 国外大模型
一文搞懂各模型擅长什么

模型那么多，到底该用哪个？这节课帮你理清思路，根据场景选对模型，不花冤枉钱。

先搞清楚：大模型能做什么

不管国产还是国外，所有大模型本质上都是"超级大脑"，能做的事都差不多——聊天、写文章、写代码、分析数据、翻译……

区别在于：每个模型的侧重点不同。有的推理能力特别强，有的中文理解特别好，有的性价比超高，有的生态最完善。

选模型的核心思路：先看场景，再看预算，最后看生态。不用追最强，只选最合适。

国外大模型

先说国外，因为大部分"最强"的模型还是来自美国公司。但要注意：大部分需要 VPN，且部分服务在国内受限。

🇺🇸 美国 Claude 4 系列（Anthropic） claude.ai

Anthropic 的当家模型，目前最新为 Claude 4 系列（含 Opus 4 / Sonnet 4 / Haiku 4）。逻辑推理极强，长文本处理优秀，对复杂任务理解深入。Opus 4 主打最强推理，Sonnet 4 平衡速度与能力，Haiku 4 主打轻量快速。适合深度分析、代码审查、长文档总结。

深度推理 代码审查 长文本分析 复杂任务拆解 Claude Code

🇺🇸 美国 GPT-5 系列（OpenAI） chatgpt.com

OpenAI 最新主力模型，最新版本 GPT-5.4（2026年初）。内置路由器，根据任务自动选择快速推理或深度思考模式。o3 系列为独立推理模型，在数学和复杂推理上表现极强。生态最完善：Codex、Sora（视频）、Operator（AI Agent）、Deep Research、ChatGPT Atlas 等全家桶全面覆盖。

GPT-5.4 o3 推理 Sora视频 Operator Deep Research API开发

🇺🇸 美国 Gemini（Google） gemini.google.com

Google亲儿子，和Google生态（搜索、Workspace、GCP）无缝集成。多模态能力突出，可以处理文字、图片、视频、音频。免费版体验已经很好，适合已经用Google全家桶的用户。

Google生态 多模态 搜索增强 免费使用

🇺🇸 美国 Grok（xAI / Elon Musk） x.com/grok

Elon Musk旗下xAI的模型，主打"反觉醒"立场和实时信息获取（接入了X平台数据）。风格直接犀利，适合需要快速获取新闻资讯、不喜欢过度"安全过滤"的使用者。

实时资讯 X平台整合 犀利风格

国产大模型

国产模型近年来进步飞速，部分领域已接近甚至超越国外水平，且在国内访问无障碍，中文理解普遍更好。

🇨🇳 中国 DeepSeek（深度求索） deepseek.com

2024-2025年最火爆的国产模型。推理能力极强（DeepSeek-R1直接对标OpenAI o1），开源+便宜，中文理解出色。特别适合需要强推理但预算有限的用户。API价格是GPT-4的几十分之一。

强推理 开源免费 性价比极高 中文 API开发

🇨🇳 中国 百度文心一言（ERNIE Bot） yiyan.baidu.com

百度基于ERNIE 4.0大模型，和百度搜索、文库、地图等生态深度绑定。中文创作能力强，政府和企业客户多。适合需要中文内容创作、且已有百度生态依赖的用户。

中文创作 百度生态 企业用户

🇨🇳 中国 阿里通义千问（Qwen） tongyi.aliyun.com

阿里自研大模型，开源力度很大（Qwen系列开源模型在全球下载量很高）。多模态能力强，支持图片、视频、代码。适合需要开源模型做二次开发、或者在阿里云上做应用开发的企业用户。

开源模型 阿里云集成 多模态 二次开发

🇨🇳 中国 腾讯混元（Hunyuan） hunyuan.tencent.com

腾讯自研大模型，和微信、腾讯文档、企业微信、腾讯云深度整合。适合已有腾讯办公生态（企业微信、TIM、腾讯会议）的企业用户，实现无缝接入。

腾讯生态 企业微信集成 企业用户

🇨🇳 中国 字节豆包（Doubao） www.doubao.com

字节跳动（抖音/TikTok母公司）出品，免费使用额度充足。界面体验好，对话流畅，适合普通用户日常使用。和抖音生态有一定整合，适合内容创作者。

免费额度 日常使用 内容创作 抖音生态

🇨🇳 中国 Kimi / Moonshot（月之暗面） kimi.moonshot.cn

主打超长上下文（支持100万-200万字），非常适合处理长文档分析、长篇小说总结、代码库理解等需要全局视野的场景。早期因为"帮我读论文"功能出圈，学生和研究人员用得多。

超长上下文 长文档分析 论文阅读 代码库理解

🇨🇳 中国 智谱清言（GLM / ChatGLM） chatglm.cn

清华大学技术背景，国产开源模型先驱之一（ChatGLM系列开源）。技术底子扎实，适合研究人员和技术开发者。GLM-4在推理和中文任务上表现均衡。

学术研究 开源模型 技术开发者

🇨🇳 中国 阶跃星辰（Step） stepchat.com

新兴国产大模型公司，Step系列模型在多模态和推理上表现亮眼。专注于多模态理解与生成，和微软有合作。适合需要尝试最新国产技术、关注前沿发展的用户。

多模态 前沿技术 推理

🇨🇳 中国 MiniMax（海螺AI） hailuoai.com

主打AI内容创作，尤其是视频生成和语音合成。和字节豆包定位类似，面向普通用户和创作者，提供免费额度，日常使用体验流畅。

视频生成 语音合成 内容创作 免费使用

根据场景选模型

不用记所有模型，只需要记住你常用的2-3个。下面的场景帮你快速定位。

学生/研究者 读论文、做研究

需要处理大量PDF和英文文献，要求理解准确、逻辑清晰。

Claude 4 Kimi（长文档） 智谱清言

开发者 写代码、API开发

需要强推理、低成本、API稳定，想自己搭应用。

Claude 4（代码审查） DeepSeek（性价比） GPT-5.4（生态）

职场人士 写邮件、做PPT、写报告

中文内容为主，需要稳定可靠，不想折腾。

DeepSeek（推理强） 豆包（免费流畅） Claude 4（深度分析）

企业用户 对接内部系统、客服、数据分析

需要稳定API、企业合规、和现有系统集成。

DeepSeek（API便宜） 通义千问（阿里云） 混元（腾讯云） Claude 4（企业版）

内容创作者 写文案、脚本、爆款内容

需要创意足、中文好、能跟上热点、免费更好。

豆包（抖音热点） 文心一言（中文创作） GPT-5.4（生态+Sora）

预算有限 想省钱、想用开源

不想花钱订阅，想自己部署或者用免费API。

DeepSeek（开源+便宜API） 通义千问（开源） Gemini（免费版强） 豆包（免费额度足）
一句话总结：
推理和代码 → DeepSeek / Claude 4 / GPT-5.4
中文日常 → 豆包 / Kimi / DeepSeek
企业生态 → 腾讯混元 / 通义千问 / 文心一言
长文档处理 → Kimi / Claude 4
省钱开源 → DeepSeek / 通义千问（开源版）

选模型的3个问题

1. 你在国内还是国外？

国内 → 优先选国产模型（DeepSeek、豆包、Kimi），访问无障碍。
国外 → Claude 4、GPT-5.4、Gemini 依然是第一梯队。

2. 你主要做什么？

推理/代码 → DeepSeek、Claude
中文内容创作 → 豆包、文心一言、DeepSeek
长文档分析 → Kimi、Claude
企业集成 → 通义千问、混元

3. 你的预算是多少？

免费 → 豆包、DeepSeek开源版、Gemini
低成本 → DeepSeek API（极便宜）
无所谓 → Claude / GPT-4o Plus
