# AI教育培训

一个轻量可运行的原创落地页项目，用于展示“AI教育培训 / AI实战训练营”方向的课程定位与招生结构。

## 启动方式

```bash
cd "项目工作区/项目 ai教育培训"
npm start
```

默认访问地址：

```bash
http://127.0.0.1:3000/
```

## 当前内容

- `public/index.html`
  - 原创单页落地页
  - 包含课程定位、痛点、模块、学习路径、FAQ、CTA
- `public/codex-first-lesson.html`
  - Codex 第一期教学提案
  - 包含问卷分析、原创预热文案、Mac/Windows 安装方法、课程大纲
- `server.js`
  - 原生 Node 静态文件服务
- `package.json`
  - 最小运行配置
